# Changelog

Todos los cambios importantes de este proyecto serán documentados en este archivo.

---

## [1.0.0] - 2026-07-03

### Añadido

- Sistema de autenticación con Firebase Authentication.
- Gestión de usuarios con roles (Administrador y Residente).
- Creación de alertas SOS para residentes.
- Gestión de avisos comunitarios por administradores.
- Creación y administración de recordatorios.
- Gestión de comunidades mediante código de invitación.
- Aplicación Web Progresiva (PWA).
- Instalación en dispositivos móviles y escritorio.
- Notificaciones locales.
- Integración con Cloud Firestore.
- Reglas de seguridad para Firestore y Firebase Storage.
- Despliegue mediante Firebase Hosting.

### Tecnologías

- Angular 20
- Ionic 8
- Firebase
- TypeScript
- SCSS
- Capacitor
